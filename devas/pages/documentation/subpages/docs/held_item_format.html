Held Item Format [Technical]

24-10-2022

To give an item with custom stats in Devas, the format of the item tag typically goes like the following:

give @s diamond_sword{Stats:{Weapon:{WeaponDmg:10, Strength:5, Health:20}}}

This line of command would give an item which has:

  • Weapon Damage: 10
  • Strength: 5
  • Health: 20

when held on mainhand.


In the first curly bracket, you would see the Stats object. Inside of it, is what the datapack reads, and evaluates the stats.

Default Equipments

Last Updated: 24-10-2022

In default, there are 7 Main equipment calculation methods for the player, which are:

  1. Weapon (Mainhand)
  2. Shield (Offhand)
  3. Helmet
  4. Chestplate
  5. Leggings
  6. Boots
  7. AbilitySlot (See more at here)

Bows

Last Updated: 24-10-2022

For bows, default stats would not work with this tag, instead you would need to put it in Weapon or Shield in order to work for Mainhand and Offhand.

Bow: {
    Power: 50,
    LoadTime: 20,
    Velocity: 3000,
    Earth: 3,
    Water: 0,
    Fire: 2,
    Wind: -3,
    Light: 2,
    Dark: 0,
    
    Mastery: {
        Earth: 3,
        Water: 0,
        Fire: 2,
        Wind: -3,
        Light: 2,
        Dark: 0
    },
    
    Defense: {
        Physical: 10,
        Earth: 3,
        Water: 0,
        Fire: 2,
        Wind: -3,
        Light: 2,
        Dark: 0,
        True: 1
    }
}

Wands

Last Updated: 24-10-2022

For wands, default stats would not work with this tag, instead you would need to put it in Weapon or Shield in order to work for Mainhand and Offhand.

Wand: {
    "Ability": 0,
    "MagicDmg": 10,
    "Earth": 3,
    "Water": 0,
    "Fire": 2,
    "Wind": -3,
    "Light": 2,
    "Dark": 0,
    "True": 1
}

Abilities

Last Updated: 24-10-2022

For abilities, SelAbil tag is for Wand Abilities. For more info on ability ids, please check here.

Ability: {
    SelAbil: 0,
    Custom: ["Dodge"]
}

Food

Last Updated: 24-10-2022

For food, The item used MUST be a warped fungus on a stick.

Food: {
    Left: 500,
    Max: 500,
    Hunger: 10
}
