In Memory of Devas - 2026

A page created to preserve the spirit, stories, and milestones of the Devas journey.

This page is created in 2026, and is about the memories of Devas.

Devas is a Minecraft Datapack that aims to create a vanilla RPG experience. Despite countless syntax changes that Microsoft made to datapacks since 2021, the project managed to keep updating to newer versions.

However, the main developers of this project have moved on to other things in their lives, and the project is now in a state of limbo. The memories page is created to commemorate the good old days of Devas, and to share the memories of the project with the community.

Devas is currently already in a playable state, and we hope that someday the project can continue to grow. In here, we will share the memories of the project, including the development process, the challenges we faced, and the fun moments we had.

We have gambled on the long run. Since the EULA does not allow developers to profit from their own creations, we'll continue to work on our other projects, to hopefully be able to fund the development of Devas in the hopefully non-distant future.

We hope that the project can continue to grow, and that it can bring joy to the Minecraft community for years to come.
