Description:
Inside the folder, you would find a total of 297 pictures inside the pic folder, named 1-297, with jpg format.

We recommend giving this puzzle a try before going to the solution, as the solution is not as complicated as you thought.


Thought Process:


Upon analyzing the pictures, it is found that some of the pictures in the folder are repeated. Therefore, we can tell that the actual image of the folder is not important.



Plans for the Flag:


Idea 1: (Dead End)

The first idea we had thought of before finding the solution, is to split the photos up to the factors of 297, notably 11 x 27, 9 x 33 and 3 x 99.

However, no specific pattern can be found after arranging the files in this manner. (rip)





Idea 2:

We then decided to start analyzing the photo data.

When we threw these pictures into autopsy, we had two main directions for finding the key.

  1. Creation Date of pictures
  2. Coordinates of pictures







Analyzing the Data:


1.  Creation Date? (Dead End)

Apparently all of the pictures have the same creation date lol, so nope.





2.  Coordinates?

When we decided to analyze the coordinates of all the pictures. It was found that even the SAME pictures have different coordinates.

With this new found knowledge, we tried to plot all the different picture coordinates into a map, hopefully seeing some sort of pattern.



[How to plot waypoints into Google Map (My Map) via Google Photos]




Unexpected Behavior


When we deployed the album of 297 photos into Google Map, we found strange patterns which seem like the flag, but didn't make any sense.

It is obvious that it is the flag, but the flag pattern is not showing up completely.





Explanation:


It appears that Google Map can only show 100 locations per layer. Therefore, we would need to split up the data set to 3 seperate albums, and then putting them in seperately.
